Domain Name System(DNS)

Definition

DNS translates domain names into IP addresses making borrowing easier and connect users to correct servers.

DNS has step-by-step process:

  • Browser Cache: Your browser checks if it already knows the IP from a previous visit.
  • Recursive Resolver: If not, it asks your Internet Service Provider (ISP).
  • Root/TLD Servers: If the ISP doesn't know, it asks global servers that handle .com, .org, etc.
  • Authoritative Nameserver: This is the final stop (like GitHub’s own DNS) that gives the exact IP address
