Web browser

Definition

Web browser is a piece of software that runs on a computer and allows one to view web pages.

Web browser has three main functions; sends requests to servers, interprets the codes, it paints pixel on the screen so as to see a website.

It is also client-sided as it handles everything the user sees and also picky sometimes. The technologies used in web browsers include:

  • HTML (Structure): The skeleton. It tells the browser what content is there (text, images, links).
  • CSS (Style): The skin and clothes. It tells the browser what colors, fonts, and layouts to use.
  • JavaScript (Logic): The brain. It tells the browser how to react when someone clicks a button or moves their mouse.
