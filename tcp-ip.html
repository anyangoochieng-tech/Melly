Transmission Control Protocol

Definition

TCP/IP is a set of protocols that helps computers to communicate to each other ensuring reliable data transfer. It works in a layered model—Application, Transport, Internet, and Network Access—where data is broken into packets, addressed, routed, and reassembled to ensure reliable, end-to-end communication

The common web application protocol in TCP/IP consists of:

  • FTP/SFTP: Used for file transfers.
  • SMTP: Used for sending email.
  • HTTPS: Secure HTTP that uses SSL/TLS for encryption.
  • ICMP: Used for diagnostics, such as the ping command.
