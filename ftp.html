File Transfer Protocol(FTP)

Definition

FTP is a protocol used to transfer files over a network.

FTP is unique because it uses two separate connections simultaneously to do its job:Command Channel (Port 21): This is the "control" line. It handles your login credentials and the commands you send, like "list files" or "delete folder" and Data Channel (Port 20): This is the "heavy lifting" line. It only opens when you actually start transferring a file (uploading or downloading).

Core commands of the FTP:

  • PUT: Uploads a file from your computer to the server.
  • GET: Downloads a file from the server to your computer.
  • CD: Changes the current folder on the remote server.
  • LIST: Shows all files and folders in the current directory.
