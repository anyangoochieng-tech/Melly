Intanet

Definition

Inranet refers to a private network that consists of web pages and resources only available within organisation so as to secure internal communication and information sharing within organisation.

An intranet uses the same technologies as the public web (HTTP, TCP/IP, and Web Browsers), but it sits behind a Firewall.

Organisations use intanets for:

  • Internal Communication: Employee news, and directories.
  • Document Sharing: Storing manuals, HR forms, and training videos.
  • Collaboration: Tools like internal wikis or project management boards.
  • Security: Sharing sensitive data that is too risky to put on the public internet.
