Web Server

Definition

A web server refers to both the physical hardware and the specialized software that stores and delivers website content (like index.html) to users over the internet. Content served can be dynamic that includes extra tools like application server or database, can also be static consisting of hardware and software that sends file to the browser for storage

A web server works as follows:

  • Request: Your browser sends an HTTP request to the server.
  • Processing: The server software receives this request, searches its storage for the requested file, and verifies you have permission to see it.
  • Response: If found, the server sends the file back as an HTTP response. If not, it sends an error like 404 Not Found.
