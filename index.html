Terminology

Definition

Web application presents tailored content based on requested parameters. It enables distribution of documents over the World Wide Web, manages concurrency, and includes role-based security.

Web application comprises of World Wide Web, TCP/IP, Web Browsers, URLs, Domain Name Server, HTTP, Internet, Extranet, FTP, HTML, Web 1.0, Web 2.0, Web 3.0, and Web 4.0.

Each of the above terminology is thoroughly elaborated on its own page.
