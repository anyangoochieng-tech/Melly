World Wide Web (WWW)

Definition

The World Wide Web refers to a system of interlinked web pages and resources providing access to information globally. It is a wide-area hypermedia information retrieval system aimed at giving universal access to a large array of documents.

Key components of the WWW include:

  • Hypertext Transfer Protocol (HTTP): The rules used to transfer data across the web.
  • Hypertext Markup Language (HTML): The standard formatting language used to create and display web pages.
  • Uniform Resource Locators (URLs): The unique address used to locate resources on the web.
