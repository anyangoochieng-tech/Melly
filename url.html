Uniform Resource Locator(URL)

Definition

An URL is an identifier for location of document on a website that comprises of domain name,protocol,port number and a path.

URL specifies protocol to use and identifies location on a website. It has two types the absolute which has full adress including domain and protocol and the relative which has a shortcut that points to the file within a project folder.

The URL also breaks due to spelling errors, case sensitivity and extensions.The components of a URL comprises of:

  • Protocol: The "language" used to transfer data. HTTP stands for Hypertext Transfer Protocol. If it has an S (https), it means the connection is Secure (encrypted).
  • Domain/IP (127.0.0.1): This identifies the server. 127.0.0.1 is a special IP called "localhost," which means the server is your own computer.
  • Port (:5500): Think of this like a specific "gate" or "door" on the server. Live Server uses port 5500 by default to send your website data to your browser.
  • Path (/tcp.html): This points to the specific file you want to see. This is the part that caused your recent error—if the path doesn't match your file name exactly, the browser gets lost.
