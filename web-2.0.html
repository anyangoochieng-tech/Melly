Web 2.0

Definition

Web 2.0 is the Read-Write web that is interactive and used for social media leading to collaboration.

Interms of technology Web 2.0 relies on: APIs that allows different apps to talk to each other, Databases whose content is pulled from a database on the fly, allowing for personalized profiles and feeds, and Mobile-First that was designed to work on smartphones and tablets, not just desktop computers.

Key characteristics of Web 2.0:

  • User-Generated Content: Instead of just reading what companies publish, users create the content.
  • Interactivity: You can "like," comment, share, and vote. The web became a two-way conversation.
  • Dynamic Pages: Sites use JavaScript and AJAX to update parts of a page without a full refres.
  • Social Connectivity: It is built around communities, networking, and collaboration.
