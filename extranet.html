Extranet

Definition

Extranet is an extension of intranet accessible to external users securing data sharing and collaboration with patners.

An extranet uses the same web technologies (HTTP, TCP/IP) you are but it requires authentication (a username and password). It creates a "secure tunnel" into a portion of a company's internal network

Real-World examples of extranet are used in companies such as supply chain, education and banking.

Importance of using extranet:

  • Efficiency: Instead of sending thousands of emails back and forth, partners can log in and see live data themselves.
  • Security: It is much safer than the public internet because every user is verified, but more flexible than an intranet because people can access it from outside the office.
  • Cost: It reduces the need for physical paperwork and manual data entry between different companies.
